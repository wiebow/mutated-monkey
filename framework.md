# Introduction #

This framework simplifies the creation of 2D Monkey games, mainly for the Desktop target.

# Details #

Current features:
  * Game
    * States
    * Transitions
  * Global Classes
    * Resource Manager

Planned features:
  * Input control
  * Audio
  * more to come...

Additional modules used:
  * AutoFit2 (http://www.monkeycoder.co.nz/Community/posts.php?topic=1500&page=1)
