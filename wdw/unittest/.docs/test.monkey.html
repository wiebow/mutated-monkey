Class Test Extends Assert
Description:A test defines a set of test methods to test.
Details:Extend TestBase to define your own tests. You MUST end the extended class name with 'Test', otherwise your test is not found.

End a method name with Before and initialize any variables/data in that method.
End a method name with After to release any permanent resources you allocated in the setup.
End a method name with BeforeClass to initialize variables/data you need for the class test.
End a method name with AfterClass to released any permanent resources you allocated in the setup.
For each test method you want to run, end the name with Test.

Any methods not using this naming scheme are ignored.
